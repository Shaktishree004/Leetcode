
# LeetCode Practice Repository

This repository is a growing collection of LeetCode solutions that I’ve organized for clear study and review. Each problem includes:

- a short explanation of the idea behind the solution
- a step-by-step approach
- implementation files in Python, Java, and C++
- complexity analysis

The goal is to keep solutions easy to read, consistent, and useful for future reference.

## Repo Structure
- Each problem has its own folder.
- Every folder includes a problem write-up and language-specific solution files.
- A reusable template for future problems is available in [PROBLEM_TEMPLATE.md](PROBLEM_TEMPLATE.md).

## How to Use This Repo
Browse a problem folder to read the explanation first, then open the implementation that matches your preferred language.

If you are adding a new solution, follow the structure in [PROBLEM_TEMPLATE.md](PROBLEM_TEMPLATE.md) so the repository stays consistent.
